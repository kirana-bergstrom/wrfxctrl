*wrfxctrl* is a website that enables users to submit jobs to the `wrfxpy <https://wrfxpy.readthedocs.io/en/latest/index.html>`_ framework for fire simulation.

wrfxctrl provides a graphical interface for simulating fires using WRF-SFIRE.
